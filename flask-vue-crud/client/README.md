## Project setup

### Flask Setup
$ mkdir flask-vue-crud
$ cd flask-vue-crud

#### Create a new directory called "server"
$ mkdir server

#### Inside "server" directory 
$ python3.7 -m venv env
$ source env/bin/activate

#### Install Flask and Flask-CORS
$ pip install Flask==1.0.2 Flask-Cors==3.0.7
